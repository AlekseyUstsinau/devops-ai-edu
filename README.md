# DevOps AI Education Repository

Welcome to the **DevOps AI Education Repository**! This repository demonstrates how to leverage AI tools like Large Language Models (LLMs) and GitHub Copilot to accelerate various phases of the DevOps lifecycle, including analysis, design, implementation, testing, and deployment.

---

## Repository Structure

The repository is organized into the following sections:

### `docs/`
Contains detailed step-by-step demo plans for each phase of the DevOps lifecycle:
- **01-analysis-and-design.md**: Demonstrates how LLMs assist in generating epics, stories, and architecture diagrams.
- **02-implementation.md**: Covers infrastructure setup, containerization, and deployment automation.
- **03-testing-and-deployment.md**: Focuses on CI/CD pipeline creation, quality gates, and monitoring.
- **04-ai-tools.md**: Explores the use of CLI-based and direct cloud access tools with LLMs.

### `.github/instructions/`
- **terraform.instructions.md**: Best practices and guidelines for writing Terraform code.

---

## Key Features

1. **AI-Assisted DevOps**:
   - Utilize LLMs and Copilot to generate infrastructure code, scripts, and workflows.
   - Automate repetitive tasks and reduce errors.

2. **End-to-End Demos**:
   - Follow detailed demo plans to understand how AI tools integrate into the DevOps lifecycle.

3. **Best Practices**:
   - Learn Terraform best practices and apply them to real-world scenarios.

## Getting Started

### Prerequisites
- **Terraform**: Install Terraform for infrastructure provisioning.
- **Docker**: Install Docker for containerization.
- **Helm**: Install Helm for Kubernetes deployments.
- **AWS CLI**: Configure AWS CLI for cloud interactions.

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/GodelTech/devops-ai-edu.git
   cd devops-ai-edu
   ```
2. Follow the demo plans in the `docs/` folder to execute each phase.
