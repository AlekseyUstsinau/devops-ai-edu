## Benchmark Questions

### Question 1
What does LLM stand for?

A) Large Language Model  
B) Linear Learning Machine  
C) Local Language Manager  
D) Long-term Language Memory  

**Correct Answer: A**

---

### Question 2
Which of the following is the primary purpose of prompt engineering?

A) Writing code for AI systems  
B) Crafting effective instructions to get desired outputs from LLMs  
C) Training new language models from scratch  
D) Debugging machine learning algorithms  

**Correct Answer: B**

---

### Question 3
What is the difference between zero-shot and few-shot prompting?

A) Zero-shot uses no examples, few-shot provides multiple examples  
B) Zero-shot is faster, few-shot is slower  
C) Zero-shot is more expensive, few-shot is cheaper  
D) Zero-shot requires training, few-shot does not  

**Correct Answer: A**

---

### Question 4
In LLM parameters, what does "temperature" control?

A) The processing speed of the model  
B) The cost of API calls  
C) The randomness and creativity of responses  
D) The number of tokens used  

**Correct Answer: C**

---

### Question 5
Which statement about tokens is correct?

A) Tokens are only complete words  
B) Tokens are units of text that affect cost and context limits  
C) Tokens are the same as characters  
D) Tokens are only used for billing purposes  

**Correct Answer: B**

---

### Question 6
What are hallucinations in the context of LLMs?

A) Visual outputs generated by the model  
B) Errors in the training data  
C) When the model generates false or fabricated information  
D) When the model becomes too creative  

**Correct Answer: C**

---

### Question 7
Which element is NOT typically part of a well-structured prompt?

A) Clear instructions  
B) Context and background information  
C) Personal opinions about the task  
D) Desired output format  

**Correct Answer: C**

---

### Question 8
What is the main benefit of providing examples in a prompt (few-shot learning)?

A) It reduces the cost of API calls  
B) It makes the prompt longer  
C) It guides the model to match the desired format and style  
D) It trains the model on new data  

**Correct Answer: C**

---

### Question 9
Which of the following is a real LLM model?

A) ChatBot-5000  
B) GPT-4  
C) SmartText Pro  
D) LanguageMaster X  

**Correct Answer: B**

---

### Question 10
Why is specificity important when writing prompts?

A) It makes prompts longer and more impressive  
B) It reduces ambiguity and helps generate more accurate responses  
C) It is required by all LLM providers  
D) It increases the token count  

**Correct Answer: B**
