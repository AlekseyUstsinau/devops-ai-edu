## Benchmark Questions

### Question 1
What is an autonomous agent in the context of AI?

A) A human operator controlling AI systems  
B) An AI system that can independently plan and execute tasks to achieve goals  
C) A monitoring tool for AI systems  
D) A type of database  

**Correct Answer: B**

---

### Question 2
Which of the following is a key characteristic of autonomous agents?

A) They require constant human supervision for every action  
B) They can only perform pre-programmed tasks  
C) They can make decisions and take actions independently  
D) They work only with structured data  

**Correct Answer: C**

---

### Question 3
What is the main difference between a chatbot and an autonomous agent?

A) Chatbots are faster  
B) Agents can plan multi-step tasks and use tools, chatbots primarily respond to queries  
C) Chatbots are more expensive  
D) There is no difference  

**Correct Answer: B**

---

### Question 4
What does the "ReAct" pattern stand for in autonomous agents?

A) Reactive Actions  
B) Reasoning and Acting  
C) Real-time Activation  
D) Response and Correction  

**Correct Answer: B**

---

### Question 5
Which component is essential for an autonomous agent to interact with external systems?

A) A graphical user interface  
B) Tools or functions it can call  
C) A physical robot body  
D) A dedicated server  

**Correct Answer: B**

---

### Question 6
What is a common challenge when working with autonomous agents?

A) They are too predictable  
B) Managing reliability, error handling, and ensuring safe actions  
C) They work too slowly  
D) They cannot process text  

**Correct Answer: B**

---

### Question 7
In the context of autonomous agents, what is a "tool"?

A) A physical device for hardware maintenance  
B) A function or API that the agent can invoke to perform specific actions  
C) A programming language  
D) A type of database query  

**Correct Answer: B**

---

### Question 8
What is the purpose of memory in autonomous agents?

A) To store user passwords  
B) To retain context and information across interactions for better decision-making  
C) To backup source code  
D) To manage network connections  

**Correct Answer: B**

---

### Question 9
Which of these is a real-world application of autonomous agents?

A) Static website hosting  
B) Automated customer support and task automation  
C) Simple file storage  
D) Basic calculator functions  

**Correct Answer: B**

---

### Question 10
What should be considered when deploying autonomous agents in production?

A) Agents will never make mistakes  
B) Safety measures, monitoring, and human oversight mechanisms  
C) They require no maintenance once deployed  
D) Cost is never a consideration  

**Correct Answer: B**
