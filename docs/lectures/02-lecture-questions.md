## Benchmark Questions

### Question 1
What is GitHub Copilot?

A) A version control system  
B) An AI-powered code completion tool  
C) A project management platform  
D) A code review automation system  

**Correct Answer: B**

### Question 2
Which company developed GitHub Copilot?

A) Google  
B) Microsoft (GitHub)  
C) Amazon  
D) Meta  

**Correct Answer: B**

---

### Question 3
What type of AI model powers GitHub Copilot?

A) Image recognition model  
B) Speech-to-text model  
C) Large Language Model trained on code  
D) Recommendation algorithm  

**Correct Answer: C**

### Question 4
Where does GitHub Copilot integrate to provide suggestions?

A) Only in web browsers  
B) Directly in code editors and IDEs

### Question 5
What is the primary way to interact with GitHub Copilot in your IDE?

A) Through voice commands  
B) By writing comments and code context

### Question 6
Which of the following is a valid use case for GitHub Copilot?

B) Generating code suggestions based on context

### Question 7
What should developers always do with code generated by GitHub Copilot?

### Question 8
